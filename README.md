# oneChat

oneChat is an in-progress chat application aiming to provide seamless messaging functionality. This project is currently under active development and is not yet complete.

## Features (Planned)
- Real-time messaging between users
- User authentication and registration
- Responsive user interface
- Message history and search
- Support for multimedia messages (images, files, etc.)
